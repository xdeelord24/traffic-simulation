Traffic Backtest

Compare Adaptive AI, External AI, and RL control at 120 vehicles/min spawn rate.

Backtest

Runs 3 min per scenario per mode at 120/min spawn, 3 trials averaged per scenario. Tests 8 road designs. Compares Adaptive AI, External AI (pressure-based), RL Agent (trained or heuristic fallback), and Static (fixed 30s cycle, typical real-life). Uses seeded RNG for reproducible results.
